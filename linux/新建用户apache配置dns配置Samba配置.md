# 新建用户

## 用户管理

新建用户，设置目录为/home/user01

useradd -d -/home/user01 -m user01

给user01设置密码

passwd user01

## 组的管理

新建一个组，gid为1000

groupadd -g 1000 stuff

创建一个账户，并设置起始组和附属组均为stuff

useradd -g stuff -G stuff user02
