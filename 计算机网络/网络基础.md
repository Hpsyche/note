# 网络的构成要素

## 设备

| 设备                       | 作用                             |
| -------------------------- | -------------------------------- |
| 网卡                       | 使计算机连网的设备               |
| 中继器（Repeater）         | 从物理层上延长网络的设备         |
| 网桥（Bridge）/2层交换机   | 从数据链路层上延长网络的设备     |
| 路由器（Router）/3层交换机 | 通过网络层转发分组数据的设备     |
| 4-7层交换机                | 处理传输层以上各层网络传输的设备 |
| 网关（Gateway）            | 转换协议的设备                   |

集线器：可以看做是多口中继器（可以认为集线器的每个端口都是一个中继器）
